💡 Insights

🏠
Poisson sous-estime l'avantage domicile de +7.9pp Prédit 40.9% pour victoire domicile, réalité 48.8%. Biais systématique que CatBoost est précisément conçu à corriger via l'encodage ordonné par équipe. Sur VPS avec plus de données, cet écart permettra un gain de RPS significatif.
🔴
Goulot critique : 0.82% des records ont un poisson_snapshot 21 081 matchs vérifiés mais seulement 172 utilisables — car predicted.poisson_snapshot n'est sauvegardé que depuis la refonte de mai 2026. Le pipeline accumule ~30-50 lignes/jour → qualité pilot attendue d'ici fin juin 2026.
📅
Zéro diversité temporelle — toutes les 172 lignes sont de mai 2026 Le modèle ne voit pas de variation saisonnière, de changements de forme long-terme, ni de schémas inter-saisons. Conséquence directe sur le RPS local (-13.3% vs baseline). Ce problème se résout naturellement en 3-6 mois d'accumulation.
🌎
Skew géographique vers les Amériques MLS + USL + Copa Libertadores + Copa Sudamericana + Brasileirao = 66 matchs (38.4%). Les équipes européennes (La Liga, PL, Serie A, Serie B) totalisent 39 matchs (22.7%). Le modèle sera initialement plus précis sur les ligues américaines.
📊
1 anomalie données — total buts négatif (-4 goals) 1 match avec realScore.home + realScore.away = -4 détecté dans la distribution. Probable corruption de données lors d'un import. Impact négligeable sur 172 lignes mais à purger via une validation SQL : WHERE CAST(json_extract(value,'$.realScore.home') AS INT) >= 0.
📐 Méthodologie & Sources

Source de données: Table SQLite kv, clé history_matches, format JSON array. Chaque record contient verified: true, realScore: {home, away}, et optionnellement predicted.poisson_snapshot.

Critère de sélection ML: Record doit avoir verified=true ET realScore non-null ET predicted.poisson_snapshot.homeWin non-null. Les records sans snapshot (anciens) sont exclus.

RPS (Ranked Probability Score): Métrique probabiliste pour classements ordonnés à 3 classes. Formule: RPS = (1/N) × Σ_i (1/2) × Σ_{k=0}^{1} (F_{i,k} - O_{i,k})². Plage [0, 1], plus bas = meilleur. Baseline Poisson PariScore: 0.2082.

Split train/val: 80%/20% stratifié sur la classe 1X2 (home/draw/away) pour préserver l'équilibre des classes dans les deux sets.

Features (16): Poisson 1X2 + over25 + over15 + btts + cs00 (7 num) · Fair probs home/draw/away (3 num) · Temporel month/hour/dow (3 num) · Catégoriels league/home_team/away_team (3 cat, encodage ordonné CatBoost natif).

Hyperparamètres 1X2: iterations=500, lr=0.05, depth=6, MultiClass, early_stopping=50, l2=3.0. Binary: iterations=300, depth=5, Logloss, early_stopping=40.
