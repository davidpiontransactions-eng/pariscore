# Sketch 003: Navbar Premium

## Design Question
La navbar actuelle a 18 onglets (Football, Tennis, CS2, MMA, NBA, WNBA, F1, CdM 2026, Roland Garros, Top Strategies, Hot Picks, Mes Paris, Guide, Sure Bets, Comparateur, Predictions, Tendances, Alertes...). Le scroll horizontal est la norme, mais est-ce la meilleure UX ?

## How to View
open .planning/sketches/003-navbar-premium/index.html

## Variants
- **A: Scroll Horizontal** — Onglets avec icones SVG inline (mask CSS, zero CDN). Barre active glow verte. Approche actuelle de PariScore.
- **B: Dropdown Categorie** — Un seul bouton "Sports" avec dropdown par groupes (Tennis/Foot/E-sport/Motorsport). Gain de place 70%.
- **C: Icones seules** — Chaque sport est une icone (14px). Le nom apparait au hover (title natif). Ultra-compact.

## What to Look For
- Decouvrabilite : un nouvel utilisateur trouve-t-il le sport qu'il cherche ?
- Efficacite : combien de clics pour aller de Football a Roland Garros ?
- Adaptabilite mobile : quelle variante fonctionne a 375px sans casser ?
