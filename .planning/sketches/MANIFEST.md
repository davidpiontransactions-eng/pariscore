# Sketch Manifest

## Design Direction
Exploration visuelle de la refonte premium dark de PariScore (charte #0b0e17 / #131722 / #00e676 / #0077ff).
Focus sur 3 composants critiques : graphique radar de comparaison, modal d'analyse tennis (20+ metriques), et barre de navigation (18 sports).

## Reference Points
- PariScore v9.7 (design actuel)
- Tennis Abstract (stats avancees)
- Chart.js / D3.js (visualisation)
- Flashscore (H2H comparison)
