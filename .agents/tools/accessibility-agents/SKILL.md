# Accessibility Agents (Collection)

This skill bundles 11 specialized WCAG 2.2 AA accessibility sub-agents from
[Community-Access/accessibility-agents](https://github.com/Community-Access/accessibility-agents).

## Sub-agents available

See the subdirectories/files in this folder for the 11 specialized agents.
Each covers a specific disability category per WCAG 2.2:
- Cognitive
- Mobile
- Screen reader
- Motor
- Visual
- Hearing
- Speech
- etc.

## Usage

Invoque l'agent approprié selon le type de handicap à auditer. Pour un audit
complet, passe en revue chacun des sous-agents.
