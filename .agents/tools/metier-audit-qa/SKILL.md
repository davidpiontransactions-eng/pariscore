# 🧪 Métier — Audit & QA Test

> **Rôle** : Vérifier la qualité, la conformité et la correction du code, des APIs et des données.

## Skills disponibles

| # | Skill | Nom d'appel | Fonction |
|---|-------|------------|----------|
| 1 | API Tester | `/agency-api-tester` | Tests API routes `/api/v1/...` — fonctionnel, performance, sécurité. Script `test-api.js` prêt à l'emploi |
| 2 | Reality Checker | `/agency-reality-checker` | Certificat basé preuves. Par défaut "NEEDS WORK" — vérifie que les features déclarées sont implémentées dans server.js / pariscore.html |
| 3 | Odds Audit | `/berkshire-odds-audit` | Pipeline audit cotes 3 étapes : Extract 15% → Verify cross-source → Verdict PASS/WARNING/REJECT |
| 4 | Playwright MCP | `/playwright-mcp` | E2E visuel via navigateur — screenshots, navigation, extraction DOM, validation responsive post-déploiement |

## Quand utiliser ce métier

- Tester les routes API → `/agency-api-tester`
- Vérifier qu'une feature est réellement terminée → `/agency-reality-checker`
- Audit quotidien des cotes → `/berkshire-odds-audit`
- Test E2E visuel, capture screenshot, validation UI → `/playwright-mcp`
