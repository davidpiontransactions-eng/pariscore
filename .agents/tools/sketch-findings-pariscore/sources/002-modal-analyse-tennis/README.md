# Sketch 002: Modal Analyse Tennis

## Design Question
Le popup d'analyse avancee (Learner Tien vs Auger-Aliassime, Halle Grass) doit afficher ELO, PowerScore, Hold%, Break%, TB Record, Deciding Set, Pressure Index, BP Ratio, Age, Win% Surface, DR Moyen, Aces, 1st Won%, Return Won%, Form Trend, H2H Record, Seed Tournoi... soit 20+ metriques. Quelle structure rend ca digeste ?

## How to View
open .planning/sketches/002-modal-analyse-tennis/index.html

## Variants
- **A: Tableau H2H optimise** — Grille 3 colonnes (P1 / Label / P2). L'approche actuelle, amelioree visuellement avec la charte dark.
- **B: Cartes 2-colonnes** — Layout par paires de cartes (Serve/Retour/Mental/Contexte). Moins dense, plus scan-friendly.
- **C: Accordeon par categories** — 4 sections pliables (Serve, Retour, Mental, Contexte). Minimal au premier regard, expansible.

## What to Look For
- Quelle variante permet de trouver l'info la plus importante en < 3 secondes ?
- L'accordeon cache-t-il trop d'information ?
- Les cartes 2-colonnes sont-elles trop "aerees" pour un usage pro ?
