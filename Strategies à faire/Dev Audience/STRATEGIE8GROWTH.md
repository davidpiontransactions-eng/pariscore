# 📈 Stratégie de Croissance PariScore (Inspirée d'Agensi)

## 1. Answer Engine Optimization (AEO)
- **Objectif :** Devenir la source citée par Claude/Perplexity.
- **Action :** Créer un fichier `llms.txt` décrivant l'API et les modèles de Poisson utilisés.
- **Format :** Titres H2 systématiquement sous forme de questions.

## 2. SEO Technique "Machine-Readable"
- **Structured Data :** Utiliser les schémas `Dataset`, `SportsEvent` et `FAQPage`.
- **Performance :** Viser un score PageSpeed de 100/100 (LCP < 1s).
- **Entity Anchoring :** Créer une page `/about` ultra-détaillée sur David (le créateur) et la méthodologie mathématique pour établir l'autorité (E-E-A-T).

## 3. Boucle de Contenu Data-Driven
- **Export GSC :** Chaque semaine, donner l'export Google Search Console à Claude.
- **Keyword Gaps :** Identifier les ligues peu couvertes par la concurrence mais recherchées.
- **CTR Optimization :** Laisser Claude réécrire les titres (Title Tags) pour maximiser les clics.

## 4. Architecture Technique
- **SSR (Server Side Rendering) :** Indispensable pour que les crawlers voient les scores en temps réel (déjà prévu dans notre architecture Zero-Dependency).
