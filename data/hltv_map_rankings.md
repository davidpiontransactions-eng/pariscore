## Usage Automatique

Ce fichier est lu par cs2Service.js pour enrichir les matchs **BO3 et BO5** :
- Onglet **MAPS** du Scout Drawer : rang mondial par carte + tendance
- Signal de valeur si ∆ winrate ≥ 20pp entre T1 et T2 sur la carte veto

### Refresh manuel (15j)

```bash
# VPS cron : 0 6 1,15 * * cd ~/pariscore && node tools/generate_map_rankings_md.js
# Ou déclencher manuellement :
node tools/generate_map_rankings_md.js
```
